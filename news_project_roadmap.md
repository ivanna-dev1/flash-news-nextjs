# 🗺 План проекту: FlashNews (Next.js News Portal)

Цей файл є єдиним джерелом істини для проекту FlashNews. План базується на досвіді попереднього проекту (Online Store), але з урахуванням особливостей Next.js та професійних підходів.

## 🛠 Стек технологій

- **Next.js** (App Router)
- **Tailwind CSS v4**
- **News API / GNews API** (для отримання реальних новин)
- **LocalStorage** (для збереження закладок)
- **Важливо:** Глибока верстка та фінальна стилізація UI виконуються в кінці, після наповнення сайту всіма основними компонентами.
