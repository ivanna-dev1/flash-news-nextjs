# 🗺 План проекту: FlashNews (Next.js News Portal)

Цей файл є єдиним джерелом істини для проекту FlashNews. План базується на досвіді попереднього проекту (Online Store), але з урахуванням особливостей Next.js та професійних підходів.

## 🛠 Стек технологій///

- **Next.js** (App Router)
- **Tailwind CSS v4**
- **News API / GNews API** (для отримання реальних новин)
- **LocalStorage** (для збереження закладок)
- **Важливо:** Глибока верстка та фінальна стилізація UI виконуються в кінці, після наповнення сайту всіма основними компонентами.

## 💡 Нотатки для навчання (Learning Notes)

- **App Router**: Розуміння різниці між `layout.js` та `page.js`.
- **Server Components**: Розуміння того, як Next.js рендерить сторінки за замовчуванням на сервері, і коли нам потрібно використовувати `'use client'` (наприклад, для `useState` у пошуку).
- **Компонентний підхід**: Розділення відповідальності, винесення логіки та констант (`INITIAL_STATES`).
- **Стійкість UI**: Завжди передбачати що зображення може бути битим, а текст не прийти, щоб дизайн "не ламався" (Флекс-бокси, заглушки).
